New Sounds

  • Added new voice lines and sound effects for champions.
  • Voice lines for Q and W abilities now have a 50% chance of playing.

Movement System Rework

  • Player hitbox now updates continuously for all movement states.
  • Dash: Cannot pass through obstacles.
  • Leap: Can jump over obstacles if the endpoint is outside them; otherwise the champion leaps to the nearest obstacle edge. Champions are immune to damage and most effects while airborne, with some exceptions.
  • Teleport: Behaves like Leap but is instant.
  • The following abilities still deal damage and apply debuffs to leaping champions: Billy's Nothung (E), BMO's Game Frame (W), Choose Goose's Electric Charge (Q), Cinnamon Bun's Ring of Frosting (E), Flame Princess's Scatter Fire (Passive), and Ice King's Hailstorm (W).

Bug Fixes & Improvements

  • Bots now correctly display in the TAB view and end game summary.
  • Disconnect buffs now apply in bot games.
  • Fixed an issue where bots would not be revealed from stealth when damaged or when attacking.
  • Improved bot AI. Bots now make smarter decisions around ability usage and positioning.
  • Improved hit detection for all projectiles. Projectile hit detection now uses a rectangle that matches the spell indicator shape.
  • Fixed an issue where skillshots that destroy on first affected target would not hit the closest one.
  • Improved minion lane pathing, targeting, and wave stacking. Fixed issues where minions would get stuck, stop moving, or stand still in lane instead of pushing.
  • Fixed an issue where only the first player to accept a party invite would successfully join the team. Subsequent accepted invites were incorrectly treated as declined.
  • Fixed incorrect buff durations on Goo Monster and Keeoth.
  • Fixed an issue where minions would move across the entire map to their correct position immediately after spawning.
  • Shortened announcer voice lines in the tutorial, reducing the time players are unable to move between prompts.
  • Added a status icon to indicate when a champion is affected by Anti-Magic Cube's special effect.
  • Corrected Ice King's Q projectile range to match the cast indicator.
  • Fixed an issue where Peppermint Butler's Q blind would persist longer than intended.
  • Fixed an issue where at the start of the match the stat menu would show base max health instead of the base + per level value.
